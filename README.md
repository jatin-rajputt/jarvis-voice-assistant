# Jarvis Voice Assistant

A secure Python-based voice assistant with system automation,
file management, logging, and ownership control.

## Features
- Voice-controlled commands
- System power control
- File & folder management
- Ownership verification (PIN)
- Action logging
- News and web search

## Security
- API keys stored using environment variables
- Logs excluded from version control

## Tech Stack
- Python
- SpeechRecognition
- pyttsx3
- OpenAI API
- MediaStack API
